# Steel Contour

Статический сайт, который можно запускать локально через расширение **Live Server** для Visual Studio Code.

## Страницы

- `index.html` — главная;
- `catalog.html` — компактный каталог выполненных навесов;
- `project-1.html` — `project-4.html` — отдельные страницы проектов;
- `services.html` — услуги и порядок работы;
- `contacts.html` — контакты и форма заявки.

## Локальный запуск

1. Откройте папку `C:\Users\workk\projects\bram-site` в Visual Studio Code.
2. Установите рекомендованное расширение **Live Server** от Ritwick Dey (`ritwickdey.LiveServer`).
3. Откройте файл `index.html`.
4. Нажмите **Go Live** в строке состояния Visual Studio Code.

Также можно нажать правой кнопкой мыши по `index.html` и выбрать **Open with Live Server**.

Сайт будет доступен по адресу:

```text
http://localhost:5500/
```

Изменения в HTML и CSS автоматически отображаются в браузере после сохранения файлов.

## Остановка сервера

Нажмите `Port: 5500` в строке состояния Visual Studio Code.
